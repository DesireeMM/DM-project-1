WHEN TO CHILL


You're busy, we get it. Us, too.

When To Chill makes navigating adult friendships a little easier by allowing you to set your regular availability for the whole group to see. Plan events with a few simple clicks for a date and time you know works for your crew. No need to spam the group chat to know when to chill.




How to use When To Chill

Creating An Account

Enter your email, first name, last name, and a password. If you like, you can opt to add a phone number.
If your email is already associated with an account, you won't be able to create a second one.
Password security is managed with argon2 hashing, but to protect yourself from security breaches, we suggest using a password unique to this site.

Once you've created an account, you'll be redirected back to the homepage to login.

Changing Your Password

You can change your password by using the form on the My Account Settings page. Please make sure your new password matches in both fields before clicking Change Password.

Adding

From your dashboard, locate the dropdown menus under Add New Availability Record. Select a day of the week, a start time, and an end time for your availability. Note that this app works best if your time blocks start and end on the hour or half hour.

Updating
  1. Click the Change Record button under the availability record you'd like to edit.
    • Alternatively, you can delete the record entirely by clicking Delete Record and confirming your action.
  2. Make changes using the dropdown menu that appears.
  3. Click the Make Changes button to save your edits.

Creating
  1. Navigate to the Create New Group page.
  2. Add in a Group Name -- this is what will display on users' dashboards.
  3. Click the Add Another Memberbutton to display a field for other users' email addresses.
    • You can add members to a group later from the group's homepage as well.
  4. Click the Create Group button when satisfied with the fields.
Updating

Newly created events will show up on your dashboard or on your My Groups page. For groups you have created, you have the option of:

  • Uploading a group photo
  • Adding new members
  • Deleting the group entirely

What if I no longer want to be a part of a group?

If you need to leave a group, click the group's Leave Group button from your My Groups page.

Creating
  1. Navigate to the Create New Event page.
  2. Add in an Event Name -- this is what will display on users' dashboards and calendars.
  3. Description, Location, Date, and Time are all optional fields here.
  4. Click the Create Event button when satisfied with the fields.
Updating

Newly created events will show up on your dashboard or on your My Events page. To update an event you are hosting:

  1. Click the event you want to update
  2. Click the Update This Event button.
    • ALternatively, you can delete the event entirely by clicking Delete This Event
  3. Make any necessary edits to your event using the form fields.
    • The best day and time for your group to hang out will be displayed here for reference.
    • If you want to view the group's full availability, click View Full Availability
  4. Click the Make Changes button to save your edits.

What if I can't make an event?

If you can't make an event you've been invited to, simply click Cannot Attend from your My Events page.
